# MS-507 Before/After Note

Before MS-507, the default cockpit opened as an equal-weight dashboard grid: metrics, provider health, create forms, runtime launch, diff, terminal, and activity all competed for first-screen attention.

After MS-507, the default cockpit opens as a workspace-first Agent Development Environment. The workspace navigator owns selection and fast switching, the agent stage makes terminal/diff/review/goal/plan the primary surface, the right inspector holds create/runtime/provider/activity context for the active workspace, and daemon/provider/MCP health plus global activity moved to the operations status bar, command palette, settings, and inspector surfaces.
